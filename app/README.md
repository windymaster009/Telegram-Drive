# Telegram Drive App

This folder contains the Tauri desktop app.

## Run Locally

From this folder:

```bash
npm install
npm run tauri dev
```

If `npm run tauri dev` says Cargo is missing, install Rust from https://rustup.rs/, restart your terminal, and verify:

```bash
cargo --version
rustc --version
```

For the full setup guide, see the root `README.md`.
